App Name:RSSReader


ABOUT:
	RSSReader is an iPhone app which parses RSS feeds and enables users to subscribe to feeds which are provided in the app. Also the user can manually subscribe to feeds in the custom category by providing a website URL or an RSS link. The user can customize the maximum number of items displayed per feed using standard settings app. 
	
	This app supports two RSS formats:RSS 2.0 and Atom 1.0. The app supports portrait orientation only. Verified on Xcode 5.1 and iPhone 5s 7.1.
